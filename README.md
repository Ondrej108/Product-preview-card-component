# Product Preview Card Component

This is a solution to the <a href="https://www.frontendmentor.io/challenges/product-preview-card-component-GO7UmttRfa">Frontend Mentor challenge</a>.

The design was based on the provided Figma/design file from the challenge.

## Live Demo
You can see the live demo of the project [here](https://ondrej108.github.io/Product-preview-card-component/).

## Description
This project features a customizable product preview card designed to display product features in a simplistic and user-friendly manner. It showcases the use of HTML, CSS, and responsive design techniques to create an interactive experience.

### Features
- Responsive design for better mobile usability
- Interactive components designed for user engagement

## How to Use
Simply clone the repository and open the `index.html` file to view the product preview card in your browser.
