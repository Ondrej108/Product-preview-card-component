# Product Preview Card

A responsive product preview card built with HTML and CSS, focused on translating a Figma-based design into a clean, pixel-accurate UI component.

Live Demo: https://ondrej108.github.io/Product-preview-card-component/

---

## Features

- Fully responsive layout (mobile and desktop)
- Pixel-accurate implementation from design
- Clean and semantic HTML structure
- Modern CSS (Flexbox, spacing, typography)
- Accessible and user-friendly UI

---

## Technologies

- HTML5 (semantic markup)
- CSS3 (Flexbox, responsive design)
- CSS Custom Properties (if used)

---

## What I focused on

- Translating Figma design into code
- Building a clean and consistent layout
- Managing spacing and typography
- Writing maintainable CSS

---

## Challenges

- Matching design spacing and proportions precisely
- Handling responsive layout between mobile and desktop
- Structuring semantic and accessible HTML

---

## What I learned

- How to convert design specifications into code
- Importance of spacing, alignment, and visual hierarchy
- Writing clean and reusable CSS
